常见问题

1. 连接超时

检查网络配置，确保 CLAUDE_CODE_DISABLE_NONESSENTIAL_TRAFFIC 已设置。

2. API Key 无效

确认 API Key 正确，且智谱账户余额充足。

3. 模型选择

智谱支持多种模型，Claude Code 会自动使用 GLM-5 模型。
