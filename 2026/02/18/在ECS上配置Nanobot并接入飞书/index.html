在ECS上配置Nanobot并接入飞书

2026-02-18

本文将介绍如何在阿里云ECS上安装配置Nanobot，并将其接入飞书机器人，实现AI助手与飞书的打通。

什么是 Nanobot？

Nanobot 是一个轻量级的 AI 助手框架，支持：

  • 多种大语言模型（Claude、智谱、OpenAI 等）
  • 多平台消息接入（飞书、Discord、Telegram 等）
  • 技能系统（Skills）扩展
  • 定时任务和自动化

配置飞书机器人

1. 创建飞书应用

  1. 访问 飞书开放平台
  2. 进入「开发者后台」→「创建企业自建应用」
  3. 填写应用名称和描述

2. 配置应用权限

在「权限管理」中开通以下权限：

权限名称 权限标识
获取用户信息 contact:user.base:readonly
接收消息 im:message
发送消息 im:message:send_as_bot
获取群组信息 im:chat:readonly

3. 获取凭证

在「凭证与基础信息」页面获取：

  • App ID
  • App Secret

4. 配置事件订阅

  1. 在「事件订阅」页面启用事件订阅
  2. 设置请求网址：http://你的ECS公网IP:8080/feishu/webhook
  3. 订阅以下事件：
    • im.message.receive_v1 - 接收消息

常见问题

Q1: 消息没有响应？

检查清单：

  1. 确认 ECS 安全组开放了 8080 端口
  2. 确认飞书事件订阅配置正确
  3. 查看日志：tail -f ~/.nanobot/nanobot.log

Q2: API 调用失败？

  1. 检查 API Key 是否正确
  2. 确认账户余额充足
  3. 检查网络是否能访问 API 端点
